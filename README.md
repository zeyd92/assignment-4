## Project Description
This is an upgraded and fully improved version of my personal portfolio website.  
It now includes new UI enhancements, mobile responsiveness, dynamic behavior, and a live deployment.  
The goal of this project is to create a polished, professional, and interactive portfolio that showcases my work effectively.

---

## Main Features
###  Core Website Features
- About Me section  
- Projects list with expandable details  
- GitHub API integration showing the latest 5 repositories (sorted by date)  
- Greeting message that dynamically updates  
- Project search bar with real-time filtering  
- Dark/Light theme (saved using LocalStorage)  
- Floating contact button with user-friendly tooltip  

---

###  New Features & Improvements
- **Full mobile responsiveness**:  
  Images resize properly, project cards stack vertically, and spacing is optimized on smaller screens.

- **Navbar refinement**:  
  The greeting message no longer overlaps with the navbar on mobile, and spacing has been fixed.

- **Dynamic greeting fix**:  
  Greeting now updates correctly without resetting or getting stuck on one name.

- **Tooltip redesign**:  
  Matches dark mode theme and displays content correctly on a single line.

- **Code cleanup**:  
  Removed unused files, unnecessary scripts, repeated JSON files, and unused logic.  
  Added simple, clean comments for important parts only.

- **Improved UI polish**:  
  Better spacing, soft card shadows, refined button styles, and more readable paragraphs.

- **New project added**:  
  Added the **Gym Workout Program Web App** project card with full description and layout.

- **Live Deployment**:  
  Website is deployed using **Vercel** for fast, stable hosting.

## How to Run (Local)
1. Download or clone the repo.  
2. Open `index.html` in any browser.  
3. (Optional): Use **Live Server** for smoother development.  
4. No backend or build tools required.
