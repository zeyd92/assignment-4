## Project Description
This is an upgraded version of my portfolio website.  
It includes API integration, dynamic features, and cleaner UI.

## Main Features
- About Me section  
- Projects with expandable details  
- GitHub API: shows latest 5 repos (sorted by date)  
- Contact form with validation  
- Dark/Light theme (saved with LocalStorage)  
- Greeting message with saved name  
- Project search filter  
- Floating contact button  

## How to Run
1. Download or clone the repo.  
2. Open `index.html` in your browser (Live Server recommended).  
3. No backend required.
